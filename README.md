### How start project

```bash
# Clone
$ git clone <https://github.com/NakaokaGabriel/starwars-api>

# Go to the project
$ cd starwars-api

# Install all depedencies
$ yarn or npm install

# Execute the project
$ npm run start or yarn start
```
